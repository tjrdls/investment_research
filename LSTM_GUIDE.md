## LSTM 모델이란?

LSTM (Long Short-Term Memory) 모델은 주식 예측에 사용되는 딥러닝 모델입니다.

```
입력 데이터
    ↓
┌─────────────────────────────────────┐
│ LSTM 브랜치                          │  기술적 지표 시퀀스 분석 (20거래일)
│ (기술적 지표 시계열 분석)            │  → 상승/하락/횡보 패턴 학습
└──────────┬──────────────────────────┘
           ↓
    ┌─────┴─────┐
    │   Fusion   │  특징 결합 및 종합
    └─────┬─────┘
           ↓
┌─────────────────────────────────────┐
│ Text 브랜치                          │  회사 재무정보 임베딩 분석
│ (회사 재무정보 텍스트 임베딩)        │  → 기업 기초 분석
└─────────────────────────────────────┘

결과: [상승 확률, 하락 확률, 횡보 확률]
```

### 모델 구성
- **LSTM Branch**: 기술적 지표 시퀀스 분석 (더블 LSTM 레이어)
- **Text Branch**: 회사 정보 임베딩 (재무정보 벡터)
- **Fusion Layer**: 두 브랜치 결합 후 3분류 (상승/하락/횡보)

## 고급 설정

### 모델 구조 변경
`train_lstm.py` 수정:

```python
# 모델 파라미터 조정
model = MultimodalStockPredictor(
    num_feat=11,        # 기술적 지표 수 (기본값 유지)
    hidden=256,         # LSTM 숨김층 크기 (기본: 128)
    text_in=1536,       # 텍스트 임베딩 차원 (고정)
    proj=64,            # 텍스트 투영 차원 (기본값 유지)
    n_cls=3,            # 분류 클래수 (상승/하락/횡보, 고정)
    n_layers=3,         # LSTM 레이어 수 (기본: 2)
    dropout=0.4         # Dropout 비율 (기본: 0.3)
)
```

### 학습 파라미터 조정
```python
model, history = train_model(
    train_loader,
    val_loader,
    epochs=100,         # 에포크 수 (기본: 50)
    lr=0.0005,          # 학습률 (기본: 0.001)
    patience=15         # Early stopping 대기 (기본: 10)
)
```
