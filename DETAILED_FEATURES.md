## 🎯 시스템 개요

이 시스템은 **한국 주식시장 전문 AI 투자 분석 플랫폼**으로, 다음과 같은 특징을 가집니다:

### 주요 특징
- **멀티모달 분석**: 기술적 지표 + 재무 데이터 + 뉴스 + AI 예측을 종합
- **실시간 데이터 수집**: pykrx, DART, NewsAPI를 통한 최신 데이터 확보
- **딥러닝 예측**: LSTM 기반 상승/하락/횡보 3분류 예측
- **LLM 종합 분석**: GPT-4 기반 전문가 수준 투자 의견 제시
- **자동화된 파이프라인**: 데이터 수집부터 최종 리포트까지 완전 자동화

### 분석 대상
- **한국 주식시장**: KOSPI/KOSDAQ 상장 종목
- **분석 기간**: 최대 3년간의 과거 데이터
- **실시간 분석**: 현재 시점 기준 최신 데이터로 분석

### 출력 결과
- **종합 투자 의견**: 매수/매도/관망 추천
- **상승/하락 확률**: LSTM 모델 기반 확률 예측
- **리스크/기회 분석**: 주요 위험 요인 및 투자 기회
- **매매 전략**: 구체적인 투자 전략 및 목표가 제시

## 📊 데이터 수집 기능

### 1. 주가 데이터 수집 (pykrx)
**기능**: 한국 주식시장 OHLCV 데이터 수집 및 관리

#### 수집 대상 데이터
- **시가 (Open)**: 거래 시작 가격
- **고가 (High)**: 거래 중 최고 가격
- **저가 (Low)**: 거래 중 최저 가격
- **종가 (Close)**: 거래 종료 가격
- **거래량 (Volume)**: 거래된 주식 수

#### 수집 기간 옵션
- **3y (기본)**: 3년 데이터 (약 730거래일)
- **1y**: 1년 데이터 (약 250거래일)
- **6m**: 6개월 데이터 (약 120거래일)
- **custom**: 사용자 지정 기간

#### 스마트 캐싱 기능
- **기존 데이터 확인**: `data/downloads/{종목코드}.csv` 파일 존재 여부 확인
- **부분 업데이트**: 누락된 기간만 추가 수집
- **중복 제거**: 날짜 기준 중복 데이터 자동 제거
- **호환성 유지**: 기존 `{종목코드}_{날짜}.csv` 형식도 지원

#### 예시 출력
```
📈 주가 데이터 수집 중 (005930)...
📂 기존 데이터 로드 완료 (650 거래일)
🔄 끝 부분 누락: 20240101 ~ 20240416
📥 누락 데이터 다운로드: 20240101 ~ 20240416
➕ 데이터 병합 완료 (45 거래일 추가)
💾 주가 데이터 저장 완료 (695 거래일)
```

### 2. 재무 데이터 수집 (DART API)
**기능**: 기업 재무제표 데이터 수집 및 분석

#### 수집 대상 데이터
- **매출액 (Revenue)**: 기업의 총 매출
- **영업이익 (Operating Income)**: 영업 활동으로 인한 이익
- **순이익 (Net Income)**: 세금 공제 후 최종 이익
- **총자산 (Total Assets)**: 기업의 총 자산
- **총부채 (Total Debt)**: 기업의 총 부채
- **총자본 (Total Equity)**: 기업의 자기자본
- **발행주식수 (Shares Outstanding)**: 유통 주식 수

#### 기업코드 매핑
- **자동 매핑**: 종목코드 → 기업코드 변환
- **캐싱**: 기업코드 목록 로컬 저장 (약 4,000개 기업)
- **실시간 조회**: DART API를 통한 최신 기업 정보

#### TTM (Trailing Twelve Months) 계산
- **최근 4분기 합산**: 가장 최근 4개 분기 데이터 합계
- **연환산**: 분기 데이터를 연간 기준으로 변환

#### 예시 출력
```
기업코드 목록 다운로드 중...
✅ 3959 개 종목 로드
📊 재무제표 데이터 수집 중 (005930)...
💾 재무제표 저장 완료 (4 기간)
✅ 완료 (4 기간)
```

### 3. 뉴스 데이터 수집 (NewsAPI)
**기능**: 종목 관련 뉴스 수집 및 호재/악재 분석

#### 수집 대상
- **종목 뉴스**: 특정 종목 관련 뉴스
- **시장 뉴스**: 경제/금융 시장 전반 뉴스

- **종합 검색**: 종목명 + 종목코드 조합
- **언어 필터**: 한국어 뉴스 우선

#### 예시 출력
```
📰 삼성전자 뉴스 수집 중...
✅ 8개 뉴스 수집 완료
🌍 시장 뉴스 수집 중...
✅ 12개 뉴스 수집 완료
```

## 📈 기술적 분석 기능

### 계산되는 지표 목록 (15개)

#### 1. 이동평균선 (Moving Averages)
- **SMA 5**: 5일 단순 이동평균
- **SMA 20**: 20일 단순 이동평균
- **SMA 50**: 50일 단순 이동평균
- **SMA 200**: 200일 단순 이동평균
- **EMA 12**: 12일 지수 이동평균
- **EMA 26**: 26일 지수 이동평균

#### 2. 모멘텀 지표 (Momentum Indicators)
- **RSI (Relative Strength Index)**: 14일 기준 상대강도지수
  - 과매수: RSI > 70
  - 과매도: RSI < 30
  - 계산식: RSI = 100 - (100 / (1 + RS))

#### 3. 추세 지표 (Trend Indicators)
- **MACD (Moving Average Convergence Divergence)**:
  - MACD 라인: EMA12 - EMA26
  - 시그널 라인: MACD의 9일 EMA
  - 히스토그램: MACD - 시그널
  - 골든크로스: MACD > 시그널 (상승 전환)
  - 데드크로스: MACD < 시그널 (하락 전환)

#### 4. 변동성 지표 (Volatility Indicators)
- **볼린저 밴드 (Bollinger Bands)**:
  - 상단 밴드: SMA20 + (표준편차 × 2)
  - 하단 밴드: SMA20 - (표준편차 × 2)
  - %B: (현재가 - 하단) / (상단 - 하단)
  - 밴드폭: (상단 - 하단) / SMA20
  - 과매수: %B > 1.0 (상단 돌파)
  - 과매도: %B < 0.0 (하단 이탈)

#### 5. 일목균형표 (Ichimoku Cloud)
- **전환선 (Tenkan)**: (9일 고가 + 9일 저가) / 2
- **기준선 (Kijun)**: (26일 고가 + 26일 저가) / 2
- **선행스팬 A**: (전환선 + 기준선) / 2, 26일 선행
- **선행스팬 B**: (52일 고가 + 52일 저가) / 2, 26일 선행

#### 6. 변동성 지표
- **변동성 (Volatility)**: 5일 수익률 표준편차
- **거래량 이동평균 (Volume MA)**: 20일 거래량 평균

### 매매 신호 생성 로직

#### 상승 신호 (Signals)
- ✅ 볼린저 하단 이탈 → 반등 가능
- ✅ MACD 골든크로스 → 상승 전환
- ✅ RSI < 30 → 과매도 반등 가능
- ✅ 20일 > 50일 → 단기 상승

#### 하락 경고 (Warnings)
- ⛔ 볼린저 상단 돌파 → 과매수
- ⛔ MACD 데드크로스 → 하락 전환
- ⛔ RSI > 70 → 과매수
- ⚠️ 20일 < 50일 → 단기 하락

#### 신호 스코어 계산
- **상승 신호**: +1 ~ +2점
- **하락 경고**: -1 ~ -2점
- **총합**: -4 ~ +4 범위

#### 예시 출력
```
📊 기술적 지표 계산 중...
✅ 지표 계산 완료 (531 거래일)
✅ 완료 (신호 1, 경고 0)
```

## 💰 재무 분석 기능

### TTM 밸류에이션 지표

#### 기본 지표
- **EPS (주당순이익)**: TTM 순이익 / 발행주식수
- **BPS (주당순자산)**: 최근 자본 / 발행주식수
- **PER (주가수익비율)**: 현재가 / EPS
- **PBR (주가순자산비율)**: 현재가 / BPS
- **ROE (자기자본수익률)**: TTM 순이익 / 평균 자기자본
- **부채비율**: 총부채 / 총자본 × 100

#### PSR (주가매출비율)
- **PSR**: 현재가 / (TTM 매출 / 발행주식수)

#### 영업이익률
- **OPM (Operating Profit Margin)**: TTM 영업이익 / TTM 매출 × 100

### 밸류에이션 판단 기준

#### PER 평가
- **저평가**: PER < 10 (성장주), PER < 15 (안정주)
- **적정가**: PER 10-20
- **고평가**: PER > 20

#### PBR 평가
- **저평가**: PBR < 1.0
- **적정가**: PBR 1.0-2.0
- **고평가**: PBR > 2.0

#### ROE 평가
- **우수**: ROE > 15%
- **보통**: ROE 10-15%
- **저조**: ROE < 10%

#### 예시 출력
```
밸류에이션 분석...
✅ PER: 13.53 PBR: 3.18 ROE: 23.53%
```

## 🤖 AI 예측 기능

### LSTM 멀티모달 모델

#### 모델 구조
```
입력 데이터
├── 기술적 지표 시퀀스 (20거래일 × 11지표)
│   ├── 가격 데이터 (OHLCV)
│   ├── 이동평균 (SMA/EMA)
│   ├── 모멘텀 (RSI, MACD)
│   ├── 변동성 (볼린저, 변동성)
│   └── 거래량 (Volume MA)
│
└── 텍스트 임베딩 (1536차원)
    └── 회사 정보 벡터화

LSTM 브랜치 ──┐
              ├── Fusion Layer ── 3분류 출력
Text 브랜치 ──┘
    ↑ 상승    ↑ 하락    ↑ 횡보
```

#### 모델 파라미터
- **LSTM 레이어**: 2층 (128 hidden units)
- **드롭아웃**: 0.3
- **텍스트 투영**: 1536 → 512 → 256 → 64
- **Fusion**: 128 + 64 → 128 → 64 → 3
- **총 파라미터**: 약 120만개

#### 학습 데이터
- **종목 수**: 10개 (시총 상위)
- **데이터 기간**: 3년 (730거래일)
- **시퀀스 길이**: 20거래일
- **예측 기간**: 5거래일 후
- **레이블 임계값**: ±2% 수익률

#### 예측 결과
```json
{
  "prediction": "📈 상승",
  "probabilities": {
    "상승": 0.65,
    "하락": 0.25,
    "횡보": 0.10
  },
  "confidence": 0.65
}
```

#### 학습 프로세스
- **배치 크기**: 32
- **에포크**: 50 (Early Stopping 적용)
- **옵티마이저**: AdamW (LR=0.001)
- **손실 함수**: Cross Entropy
- **정확도**: ~65-70%

#### 예시 출력
```
🤖 LSTM 모델 학습 시작
📊 LSTM 학습용 데이터 수집
├─ 삼성전자 (005930)... ✅ 730 거래일
├─ SK하이닉스 (000660)... ✅ 730 거래일
```

## 🔗 API 및 의존성

### 외부 API 서비스

#### 1. OpenAI API
- **사용 목적**: 텍스트 임베딩, LLM 분석
- **API 엔드포인트**:
  - `embeddings`: 텍스트 벡터화 (text-embedding-ada-002)
  - `chat/completions`: 종합 분석 (gpt-4o-mini)
- **요금**: 토큰 기반 과금
- **속도 제한**: RPM 3,000 / TPM 150,000

#### 2. DART API (금융감독원)
- **사용 목적**: 기업 재무제표 데이터
- **API 엔드포인트**:
  - 기업코드 목록: `corpCode.xml`
  - 재무제표: `fnlttSinglAcnt.json`
- **제한**: 분당 100회, 일일 10,000회
- **데이터**: 실시간 반영 (분기/연간 보고서)

#### 3. NewsAPI
- **사용 목적**: 뉴스 데이터 수집
- **API 엔드포인트**: `everything`
- **제한**: 일일 100회 무료
- **커버리지**: 70,000개 뉴스 소스

#### 4. pykrx (한국거래소)
- **사용 목적**: 주가 데이터 수집
- **제한**: 없음 (공공 데이터)
- **데이터**: 실시간 주가 정보

### 내부 의존성

#### 데이터 처리
- **pandas**: DataFrame 기반 데이터 처리
- **numpy**: 수치 계산 및 배열 연산
- **scikit-learn**: 데이터 정규화 (StandardScaler)

#### 머신러닝
- **PyTorch**: 딥러닝 모델 구현
- **torchvision/torchaudio**: PyTorch 생태계

#### 유틸리티
- **python-dotenv**: 환경변수 관리
- **requests**: HTTP API 호출
- **datetime/timedelta**: 날짜 시간 처리

## ⚠️ 제한사항 및 주의사항

### 데이터 제한사항

#### 1. 주가 데이터
- **거래일 기준**: 주말/공휴일 제외
- **최대 기간**: 3년 (약 730거래일)
- **실시간성**: 전일 종가 기준
- **정정 데이터**: 기업 액션(배당, 분할) 반영

#### 2. 재무 데이터
- **보고서 기준**: 분기/연간 보고서
- **지연 반영**: 보고서 제출 후 1-2일 소요
- **표준화**: K-IFRS 기준
- **커버리지**: 상장사 전체 (약 4,000개)

#### 3. 뉴스 데이터
- **실시간성**: 24시간 내 뉴스
- **언어**: 한국어 우선, 영어 보조
- **품질**: 검증된 뉴스 소스만
- **개수 제한**: 종목당 최대 10개

### 모델 제한사항

#### 1. LSTM 모델
- **학습 데이터**: 시총 상위 10종목
- **예측 기간**: 5거래일 후
- **정확도**: ~65-70% (백테스트 기준)
- **과적합**: Early Stopping 적용

#### 2. LLM 분석
- **토큰 제한**: 입력 8,000토큰, 출력 800토큰
- **언어**: 한국어 전문
- **편향성**: 학습 데이터 기반
- **확실성**: 70-90% 신뢰도 범위

### 시스템 제한사항

#### 1. 성능
- **학습 시간**: 3-30분 (GPU/CPU)
- **분석 시간**: 30-60초
- **메모리 사용**: 2-8GB
- **저장공간**: 500MB-2GB

#### 2. 안정성
- **API 의존성**: 외부 서비스 장애 시 대체 로직
- **네트워크**: 인터넷 연결 필수
- **캐싱**: 로컬 데이터 저장으로 속도 최적화

### 투자 관련 주의사항

#### 1. 법적 고지
- **참고용**: 투자 조언이 아님
- **책임**: 모든 투자 결정은 투자자 본인
- **리스크**: 주식 투자는 원금 손실 가능
- **세금**: 투자 수익에 대한 세금 책임

#### 2. 분석 한계
- **과거 데이터**: 과거 성과가 미래 보장하지 않음
- **시장 리스크**: 경제/정치 이벤트 예측 불가
- **모델 정확도**: AI 예측은 확률적
- **실시간성**: 전일 데이터 기반

#### 3. 사용 권장사항
- **다각적 검토**: 여러 분석 방법 활용
- **리스크 관리**: 분산 투자 권장
